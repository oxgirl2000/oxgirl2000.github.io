Building a Wireless Controller with Node.js and p5

Prerequisites:

  • Install Node.js
  • Install Visual Studio Code

Settting up

Start by creating a template folder. With Visual Studio Code open, drag the folder from Finder/File Explorer into the sidebar.

At the top of the sidebar, press the icon the create a new file and name it server.js.

Navigating the terminal

Usually, working with Node.js meant that you had separate windows open for your editor to edit your code, your terminal to run your code, and a browser to see the final product. However, Visual Studio Code combines the editor and terminal (along with native syntax support for Node.js!) which makes it a lot easier to work with Node than without.

In your server.js file, add some code to send a message to the log.

console.log("Hello world!")

Now run the code by running run server.js.

NPM

Thes best part of Node.js is that it allows us to use a ton of packages found in the Node Package Manager.

npm init
npm install express --save

Server - Express

Express is a lightweight communication framework. Find the API here.

let express = require('express'); let app = express(); let server = app.listen(3000);

Now when you run the server, you can open the page on your browser by going to localhost:3000/
